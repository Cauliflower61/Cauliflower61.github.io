Core Algorithm

Multi-condition Controllable Human Video Generation

Pose-guided and reference-driven human video generation based on Wan2.2-5B and Wan2.1-14B. The project supports pose sequence control, reference image conditioning, first-frame conditioning, long-video generation, and multi-person control.

Pose2Video Wan2.2-5B Wan2.1-14B Pose-CFG Long Video Multi-person Control DMD 4-step

Overview

Project Summary

Goal

Build a controllable human video generation algorithm driven by multiple conditions, including pose sequences, reference images, first frames, and spatial identity cues.

My Role

Responsible for data construction, automatic annotation pipeline, Pose2Video model adaptation, Pose-CFG stability strategy, long-video generation, multi-person control, and DMD distillation.

Pipeline

Data and Annotation Pipeline

The training data is built from 9 open-source datasets and internal data. Metadata-based filtering is used to select high-quality samples for training.

01. Large-scale Data Pool

Built a 10M-level human video data pool from OpenHumanVid, MEAD, CECSL and other open-source datasets, together with internal data.

02. Metadata-based Filtering

Selected around 1.6M high-quality samples based on resolution, bitrate, number of persons, and other metadata.

03. Automatic Annotation

Integrated NLF for body pose, InsightFace for face information, WiLor for hand keypoints, Qwen3-VL for prompt annotation, and GroundSAM2 for person tracking and segmentation.

Method

Key Technical Designs

Pose-CFG

Proposed a Pose-CFG strategy to enhance pose guidance, improve temporal stability, and reduce motion artifacts, structural distortion, and local visual defects.

Long Video Generation

Supported multi-segment generation and stitching, enabling about 1-minute long human video generation while reducing color drift and style inconsistency.

Multi-person Control

Used bounding boxes to represent different identities and spatial locations, enabling identity-aware and position-aware multi-person control.

DMD Distillation

Applied DMD distillation to compress the generation model to 4-step inference, improving inference efficiency while maintaining generation quality.
