原生 PyTorch 分布式训练与蒸馏框架

面向视频生成模型构建的原生 PyTorch 训练框架，不依赖 Accelerate、DeepSpeed、Diffusers 等高层封装，支持 SFT、LoRA、DMD、FSDP、序列并行、梯度检查点和 EMA。

Native PyTorch FSDP Sequence Parallel Gradient Checkpointing EMA SFT / LoRA / DMD
框架概览

项目目标

构建不依赖高层封装的原生 PyTorch 训练框架，保留对训练逻辑、显存、通信和状态恢复的底层控制能力，降低新模型和新任务的接入成本。

支持任务

  • Wan2.1 / Wan2.2 视频生成模型训练
  • SFT 与 LoRA 微调
  • DMD 蒸馏训练
  • 多卡分布式训练与长周期训练恢复

分布式能力

基于 PyTorch 原生 FSDP、梯度检查点和 EMA，并基于 yunchang 实现序列并行，支持大规模视频生成模型的多卡训练与显存优化。

可观测性

支持训练各阶段耗时统计、显存监控、日志记录、断点续训和训练状态恢复，提升长周期训练任务的可维护性。
