Infrastructure

Native PyTorch Distributed Training and Distillation Framework

A native PyTorch framework for large-scale video generation model training and distillation, designed without relying on high-level wrappers such as Accelerate, DeepSpeed, or Diffusers.

PyTorch FSDP Sequence Parallel Gradient Checkpointing EMA SFT / LoRA / DMD Checkpoint Resume

Overview

Project Summary

Goal

Build a controllable and extensible training framework for Wan-series and other video generation models, supporting SFT, LoRA, and DMD training.

Design Principle

Implement the training system with native PyTorch to preserve low-level control over training logic, distributed strategy, memory usage, checkpointing, and state recovery.

Features

Core Capabilities

Factory and Registry

Unified datasets, models, optimizers, and schedulers through factory functions and registry mechanisms, enabling plug-and-play component management.

Training Modes

Supported SFT, LoRA fine-tuning, and DMD distillation for Wan2.1, Wan2.2, and related video generation models.

Distributed Training

Integrated FSDP, sequence parallelism, gradient checkpointing, and EMA to support stable and memory-efficient multi-GPU training.

Monitoring and Recovery

Added training-time profiling, memory monitoring, checkpoint resume, and state recovery to improve observability and robustness for long-running training tasks.
