Infrastructure

Native PyTorch Distributed Training and Distillation Framework

面向视频生成模型的原生 PyTorch 分布式训练与蒸馏框架，支持 Wan2.1、Wan2.2 等模型的 SFT、LoRA、DMD 训练，以及 FSDP、序列并行、梯度检查点和 EMA。

Native PyTorch FSDP Sequence Parallel Gradient Checkpointing EMA SFT / LoRA / DMD

Overview

项目概览

Project Goal

构建一个不依赖 Accelerate、DeepSpeed、Diffusers 等高层训练封装的原生 PyTorch 训练框架， 保留对训练逻辑、显存、通信和状态恢复的底层控制能力。

Supported Tasks

  • Wan2.1 / Wan2.2 视频生成模型训练
  • SFT 与 LoRA 微调
  • DMD 蒸馏训练
  • 多卡分布式训练与长周期训练恢复

Architecture

框架结构

Registry + Factory

采用工厂函数与注册机制统一管理数据集、模型、优化器、学习率调度器等训练组件， 降低新模型、新任务和新训练策略的接入成本。

Abstract Pipeline

Dataset / Model / Optimizer / Scheduler
                ↓
        Registry + Factory
                ↓
             Trainer
                ↓
 FSDP / Sequence Parallel / Checkpoint / EMA / DMD

Capabilities

核心能力

Distributed Training

基于 PyTorch 原生能力实现 FSDP、梯度检查点和 EMA，并基于 yunchang 实现序列并行， 支持大规模视频生成模型的多卡训练与显存优化。

Monitoring and Recovery

支持训练各阶段耗时统计、显存监控、日志记录、断点续训和训练状态恢复， 提升长周期训练任务的可观测性、稳定性和可维护性。

Training Modes

统一支持 SFT、LoRA 和 DMD 蒸馏训练，使不同模型和不同训练范式可以复用同一套训练入口。

Engineering Control

相比高层封装框架，原生 PyTorch 实现便于调试训练细节、定位显存瓶颈、控制通信策略与状态保存逻辑。
