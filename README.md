# AIGC Video Generation Portfolio

This is a GitHub Pages portfolio template for AIGC video generation projects.

## Structure

```text
.
├── index.html
├── style.css
├── projects/
│   ├── human-video.html
│   ├── training-framework.html
│   └── video-erasure.html
└── assets/
    ├── images/
    └── videos/
```

## Pages

- Home: `index.html`
- Project 01: `projects/human-video.html`
- Project 02: `projects/training-framework.html`
- Project 03: `projects/video-erasure.html`

## How to use

1. Upload all files to your `username.github.io` repository.
2. Enable GitHub Pages from `Settings → Pages`.
3. Replace placeholders with your own public or synthetic demo videos.
4. Do not upload proprietary data, internal code, model weights, or user materials.
